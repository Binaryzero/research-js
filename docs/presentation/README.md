# Presentation

Interactive HTML presentation for the Extension Security Analyzer project.

## Files

| File | Description |
|------|-------------|
| `presentation.html` | Standalone HTML presentation (open in any browser) |

## Usage

Open `presentation.html` directly in a browser — no server required. The presentation is fully self-contained with inline CSS and JavaScript.

## Navigation

- Arrow keys or click to advance slides
- Escape to see slide overview
- Press `?` for keyboard shortcuts

## Maintenance

| Trigger | Action |
|---------|--------|
| Major project changes | Regenerate presentation with updated content |
| New stakeholder audience | Create audience-specific version |
